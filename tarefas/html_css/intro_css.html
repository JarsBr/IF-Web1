Oque é CSS (Cascading Style Sheets)

Os arquivos CSS são folhas de estilos, que são nada mais nada menos que uma coleçao de regras de formatação que podem ser aplicadas a um documento HTML, que serve para estilizar o documento, assim controlando a estética do site.

  • HTML controla a estrutura da pagina
  • CSS determina sua aparência

Formas de aplicar CSS no documento:

  • Inline - Estilo colocando dentro da tag especifica no documento HTML
  • Incorporado - Estilo dentro da tag <style> no <head> do documento HTML
  • Folha de estilo externa - Estilo dentro de um documento .css, e usando <link rel="stylesheet" href="nome_arquivo.css"> dentro do <head> no HTML

A ordem de prioridade dos estilos é de cima para baixo, ou seja, os estilos mais abaixo podem 'sobrescrever' os estilos mais acima.

Exemplo regra CSS

p.centro {
 text-align: center;
 color: green;
 font-size: 25px;
}

  • Seletor: Elemento(tag) p
  • Declarações: text-aling: center; font-size: 25px; color: green;
  • Propriedades: text-aling color font-size

Tipos seletores

Seletor Universal: Representa todos os elementos
Representa por *

* {
 color: green;
}

Seletor de Tipo(elemento): Representa todos de um determinado elemento
Representa por <tag>

p {
 color: red;
}
h1 {
 color: yellow;
}

Seletor de classe: Aplica estilo a todos elementos que pertecem a aquela class.
Representa por .

.verde {
 color: green;
}
h1.verde {
 color: green;
}

No .verde aplica a cor verde a qualquer elemento que possua a class="verde"
No h1.verde aplica a cor verde apenas a elementos h1 que possuem a class="verde"

Seletor de ID: Aplica estilo a elemento com o atributo id definido
Representa por .

#corAzul {
 color: blue;
}
h1#corAzul {
 color: blue;
}

No .verde aplica a cor azul a qualquer elemento que possua o id="corAzul"
No h1.verde aplica a cor azul apenas a elementos h1 que possuem a id="corAzul"
